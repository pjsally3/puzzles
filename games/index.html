How To Play

  • Drag a letter from the legend (right box) onto a node.
  • If correct, all occurrences of that letter are revealed.
  • If wrong, it returns to the legend and your error count increases (unless that node’s letter is already revealed somewhere).
  • Hover a node to highlight all edges in/out and see how many times it’s used.
  • Hint reveals a random hidden letter (counts as a hint).
  • Timer starts each puzzle.
